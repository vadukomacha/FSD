# Week 5 - Node.js + Express (Basic Routing)

## Tech Stack
- Node.js
- Express
- HTML pages

## How This Project Was Created
This project was created manually as a basic Express server.
No frontend framework was used.

Main idea:
- Build a Node.js server with Express
- Serve static HTML pages using sendFile
- Add basic GET/POST route handling

## Project Structure
- server.js: Express server and routes
- index.html, login.html, register.html, welcome.html: pages
- package.json: Express dependency

## Install
1. Open terminal in this folder
2. Run:

```bash
node -v
npm install node
npm install express
```

## Run
1. Run:

```bash
node server.js
```

2. Open:
- http://localhost:3636/

## Notes
- Server port is 3636.
- This week is focused on routing and page navigation.
