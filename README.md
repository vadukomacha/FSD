## How To Download Only One Folder From GitHub
If you want to download a single folder (not full repository):

1. Open the repository on GitHub.
2. Open the folder you want (example: week10).
3. Copy the URL from browser address bar.
4. Open this website:
   - https://download-directory.github.io/
5. Paste the copied folder URL.
6. Click Enter / Download.
7. A ZIP file of that folder will be downloaded.

## Important Notes
- This works for public GitHub repositories.
- For private repositories, this method may not work without proper access.
