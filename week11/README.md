## How This Project Was Created
1. Open terminal in this folder
```bash
python --version
```
2.
```bash
py -m pip install --user virtualenv
```
3.
```bash
py -m venv venv
```
4.
```bash
.\venv\scripts\activate
```
5.Install Django:
```bash
pip install django
```
6.
```bash
py -m django --version
```
7.
```bash
django-admin startproject csef
```
8.
```bash
cd csef
```
9.Run:
```bash
py manage.py runserver
```
