## How This Project Was Created
Create a folder and open in cmd:
1.
```bash
npm install -g @angular/cli
```
2. 
```bash
ng new foldername
```
->select y
->select css

3.Create header,profile,home:
```bash
ng g c header
ng generate component home
ng g c profile
```

4.Run :
```bash
ng serve --o
```
