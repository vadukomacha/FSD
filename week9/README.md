# Week 9 - Angular (SPA)

## Tech Stack
- Angular (Standalone Components)
- TypeScript
- Angular Router
- HTTP client usage in Profile page

## How This Project Was Created
This project was created using Angular CLI.

Creation command pattern used:

```bash
npx @angular/cli new spa --directory . --standalone --routing --style=css
```

Then app files were organized under src/app, routing was connected, and duplicate root copies were removed.

## Install
1. Open terminal in this folder
2. Run:

```bash
npm install
```

## Run
1. Start development server:

```bash
npm start
```

2. Open the URL shown in terminal (usually):
- http://localhost:4200/

## Build

```bash
npm run build
```

## Notes
- Routes include Home and Profile.
- Profile uses HTTP API data.
