# Week 10 - React (Vite)

## Tech Stack
- React
- Vite
- React Router
- React Bootstrap + Bootstrap

## How This Project Was Created
Create a folder and open cmd.

1.
```bash
npm install -g npx
```
2.
```bash
npm install -g create-react-app
```
3.
```bash
npx create-react-app spa1
```
4.
```bash
cd spa1
```
5.Run:
```bash
npm start
```
